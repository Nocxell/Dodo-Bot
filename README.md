## ㅤㅤㅤㅤㅤㅤㅤㅤㅤㅤㅤㅤㅤㅤInstallation
It is installed very simply:
```python
git clone https://github.com/Nocxell/Dodo-Bot
cd Dodo-Bot
pip install -r requirements.txt
```
Before launching, do not forget to specify the token in the file main.py

Now run the script:
```python
python main.py
```
